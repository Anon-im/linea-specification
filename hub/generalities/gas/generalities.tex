We impose the following general constraints on gas columns:
\begin{enumerate}
        \item the following columns are hub-stamp-constant
		\begin{multicols}{3}
			\begin{enumerate}
				\item \gasExpected
				\item \gasNext
				\item \gasActual
				\item \gasCost
				\item \refund
				\item $\refund\new$
			\end{enumerate}
		\end{multicols}
\end{enumerate}
