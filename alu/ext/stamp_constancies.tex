As always we say that a column $\col{X}$ is \textbf{stamp-constant} if it satisfies
\[
	\extStamp_{i} = \extStamp_{i - 1}
	\implies
	\col{X}_{i} = \col{X}_{i - 1}.
\]
We impose stamp-constancy on the following columns 
\begin{multicols}{5}
\begin{enumerate}
	\item $\iArgOneHi$
	\item $\iArgOneLo$
	\item $\iArgTwoHi$
	\item $\iArgTwoLo$
	\item $\iArgTreHi$
	\item $\iArgTreLo$
	\item $\iResHi$
	\item $\iResLo$
	\item $\iInst$
\end{enumerate}
\end{multicols}
