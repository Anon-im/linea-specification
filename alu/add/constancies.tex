As per usual we declare a column \col{X} to be \textbf{stamp-constant} if it satisfies
\[
	\If \addStamp_{i + 1} \neq 1 + \addStamp_{i} ~ \Then \col{X}_{i + 1} = \col{X}_{i}.
\]
We impose stamp-constancy onto the following columns:
\begin{multicols}{4}
	\begin{enumerate}
		\item $\argOneHi$
		\item $\argOneLo$
		\item $\argTwoHi$ 
		\item $\argTwoLo$
		\item $\resHi$
		\item $\resLo$
		\item $\INST$
		\item $\maxCt$
		% \item[\vspace{\fill}]
	\end{enumerate}
\end{multicols}
