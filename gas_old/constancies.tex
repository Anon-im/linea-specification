As usual we declare a column \col{X} to be \textbf{counter-constant} if it satisfies
\[
	\If \ct_{i} \neq 0 ~ \Then \col{X}_{i} = \col{X}_{i - 1}
\]
We impose \textbf{counter-constancy} on the following columns:
\begin{multicols}{3}
\begin{enumerate}
	\item \gasActual{}
	\item \gasCost{}
	\item \oogx{}
\end{enumerate}
\end{multicols}
