We say that a column \col{X} is \textbf{log-constant} if it satisfies
\[
	\If \absLogNum_{i} \neq 1 + \absLogNum_{i - 1} ~ \Then \col{X}_{i} = \col{X}_{i - 1}.
\]
We require that the following columns be log-constant:
\begin{multicols}{3}
	\begin{enumerate}
		\item \logDataSize{}
		\item \logLogsData{} (\trash)
	\end{enumerate}
\end{multicols}
